Custom Input Sampler.


Usage:

<dst-input></dst-input> Possible attributes:
  • size --can be lg, md, and sm, the default is sm--
  • placeholder --a hint text--
  • label-left --a text that will appear at the left of the input--
  • label-top --a text that will appear on the top of the input--
  • max --a maximum number of characters--
  • valid_input --a regex to validete the input--
